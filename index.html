Introduction

This page demonstrates how to use CSS selectors and the box model to style a simple web page. We apply margins, padding, borders, and background colors to make elements visually distinct.

The CSS Box Model

The box model consists of four main parts: content, padding, border, and margin. These control the spacing and size of elements on the page.
